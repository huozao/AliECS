项目简介

项目当前定位为配方相关业务入口聚合：后续承载配方查询、需求管理、库存查询和管理后台，技术架构保持 public-web / admin-ui / backend-api / postgres。

权限机制：普通业务入口公开；“个人板块”需要登录且具备 personal 权限。

登录与权限
