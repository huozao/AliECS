项目简介

该项目定位为“配方相关业务入口聚合平台”。当前阶段用于承载配方查询、需求管理、库存查询与管理后台入口， 并保持与现有技术架构一致：public-web / admin-ui / backend-api / postgres。未实现模块将明确标记为“预留扩展”或“占位入口”。
